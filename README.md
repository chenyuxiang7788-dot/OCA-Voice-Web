# OCA Voice Web

无需安装 App 的一对一网页语音通话网站。

## 本地运行

需要 Node.js 18 或更高版本：

```bash
npm install
npm start
```

打开 `http://localhost:3000`。

## 正式部署

部署到支持 Node.js 与 WebSocket 的平台，启动命令为 `npm start`。

正式网站必须使用 HTTPS，否则手机浏览器通常不会允许麦克风。

## 使用方法

1. 第一个人进入网站，点击“加入通话”并允许麦克风。
2. 点击“复制邀请链接”，发给朋友。
3. 朋友打开链接并点击“加入通话”。
4. 双方连接成功后即可语音。

## 限制

当前只使用 STUN，没有 TURN。部分校园网、公司网或严格移动网络可能无法建立点对点连接。

服务器只交换 WebRTC 连接信息，不保存语音。
